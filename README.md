## Step_Academy_PHP_Small_Application_For_User_Upload_Photo_With_Gallery_LogIn_LogOut_Registration_Lab6

### Discription:
````
Upon registration, each new user will be provided with a personal account where they can store their uploaded photos.
The program utilizes cookies to identify the user's corresponding folder for data storage and retrieval.
During the registration process, the user's name, email address, and hashed password are securely stored.

Once successfully registered, users gain access to their account and can view the photos they have uploaded.
To ensure privacy and security, the program automatically deletes the cookies associated with the user's session upon leaving the application.
The lifespan of these cookies is set to 2 hours. After this time period, the program switches to a login mode,
requiring users to log in again to access their account and photos.
This measure adds an additional layer of protection and prevents unauthorized access to user data.

Furthermore, specific access restrictions are implemented to enhance security.
If a user is not registered or logged in, they are unable to navigate to the home page, upload photos, or access the photo gallery.
Similarly, if a user is already logged in, they are prevented from accessing the registration link to prevent duplicate accounts.

By implementing these measures, the program offers a user-friendly and secure environment for individuals to register,
store their photos, and conveniently access them within their personal account.
These safeguards protect user data and ensure that only authorized users can interact with the application's features.
````
